STEP 1 — HLS AES-128 ENCRYPTION (SERVER-SIDE)
===============================================================================

  Instead, in the HLS playlist set EXT-X-KEY URI to my backend endpoint:
  URI="/api/stream/<videoId>/key?kid=<kid>&t=<keyToken>"

Implementation detail:
- Use ffmpeg with key info file:
  key_info_file contains:
   line1: key-uri (our backend key endpoint URI placeholder)
   line2: local path where key file is written during encryption
   line3: key hex (optional)
- Store encrypted segments + playlists in B2 private bucket.
- Store encryption metadata in DB:
